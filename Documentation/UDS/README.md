# Uniform Data System (UDS) Manual

`UDS_2025_Manual.txt` is the full text of HRSA's *2025 Uniform Data System
Manual: Health Center Data Reporting Requirements* (reports due February 15,
2026), extracted from the official PDF. It documents what FQHCs (Federally
Qualified Health Centers) must report annually to HRSA's Bureau of Primary
Health Care: patient characteristics, services provided, clinical quality
measures, utilization, costs, and revenues.

Reference this file when implementing or reviewing FQHC/UDS-related
reporting features in OpenEMR (e.g. UDS clinical quality measure
calculations, visit/encounter counting rules, sliding fee scale and patient
characteristic reporting) to confirm reporting definitions match HRSA's
current requirements.

Source: HRSA Bureau of Primary Health Care, OMB control number 0915-0193.
